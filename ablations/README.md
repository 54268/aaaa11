# 消融实验说明

本目录保存当前开放集 SEI 方案的消融实验结果。

当前消融结果分成两套口径：

1. 对外展示主表  
用于 PPT / 论文正文，突出真正需要展示的模块递进效果。

2. 内部参考结果  
用于自己检查方法细节，例如“去掉冲突保护”是否真的影响最终性能。

## 三、当前结论

当前最稳妥的实验结论是：

1. 关键边界样本筛选是有效的。
2. 原型校准比 OpenMax 更关键。
3. OpenMax 在 Oracle 上有辅助作用，但不是主导因素。
4. 冲突保护目前没有体现出稳定收益，更适合作为实现细节，而不适合作为单独创新点强调。
