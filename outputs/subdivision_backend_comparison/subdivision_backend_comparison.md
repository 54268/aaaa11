# 未知类细分方法对比

本表对应论文候选图 `figures/backend_comparison.png`。当前细分对比不再展示 GMM 内部后端消融，而是按方法层面对比 K-means、HDBSCAN、OpenRFI 和 PCBM (ours)。

- K-means：基于质心距离的划分式聚类。
- HDBSCAN：层次密度聚类，不需要固定 `eps`，低密度样本会被视为噪声或不确定样本。
- OpenRFI：先用 KMeans 形成过完备原型，再在原型相似图上做谱聚类，并将样本回填到最近原型所在的簇。
- PCBM (ours)：本文完整未知类细分流程，包含原型竞争边界建模、双重校准、K+m 冗余候选和不确定样本过滤。

## Oracle

| 方法 | NMI | ARI | 匈牙利准确率 | 覆盖率 |
| --- | ---: | ---: | ---: | ---: |
| K-means | 0.852434 | 0.832207 | 0.921844 | 0.967083 |
| HDBSCAN | 0.684820 | 0.425885 | 0.487485 | 0.699167 |
| OpenRFI | 0.931928 | 0.909938 | 0.930196 | 0.850000 |
| PCBM (ours) | 0.998816 | 0.999263 | 0.999718 | 0.886292 |

Oracle 上，HDBSCAN 在当前特征分布下只解析出少量稳定簇；OpenRFI 的原型图谱聚类优于通用聚类基线，但覆盖率低于 PCBM。
